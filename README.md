# RparaCientistasSociais
Curso "R para Cientistas Sociais" - Iniciante.
Disponibilizo aqui um curso para usuários iniciantes da linguagem R, a partir do RStudio, para Cientistas Sociais.
Neste repositório teremos um arquivo .csv do ESEB 2022 para que possamos fazer as primeiras análises. Temos também o questionário, que serve como dicionário para análisar os dados.
Temos aqui também o script completo para a aula.
